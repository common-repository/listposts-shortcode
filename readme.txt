ListPosts Shortcode is a shortcode that adds a highly customized list of blog posts anywhere on their site.

== Description ==

ListPosts Shortcode offers bloggers a massive amount of control over the appearance and functionality of an otherwise simple list of posts. It is a shortcode featuring loads of innovative parameters.

* [Demo](http://listposts.lanexa.net/)
* [Documentation](http://listposts.lanexa.net/listposts-readme.txt)
* [Download](http://listposts.lanexa.net/listposts.zip)

= Main Features =

* Simple default shortcode behaves as expected, i.e. "[listposts]"
* Add parameters to customize appearance & functionality 
* Unlimited usage per page

== Installation ==

ListPosts Shortcode works like any standard Wordpress plugin, and requires zero configuration.

Whenever installing or upgrading any plugin, or even Wordpress itself, it is always recommended you back up your database first!

= Installing =

1. If installing, go to Plugins > Add New in the admin area, and search for ListPosts.
2. Click install, once installed, activate and you're done!

= Utilization = 
Once installed, insert the shortcode into the HTML editor of a page or post, i.e. "[listposts]". For a complete guide to customization, please visit the plugin site at http://listposts.lanexa.net

= Upgrading =

1. When upgrading, visit the plugins page in your admin area, scroll down to events manager and click upgrade.
2. Wordpress will help you upgrade automatically.
